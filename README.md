# video-transcriber 🎬→📄

Транскрибация видео и аудио в текст. Закинул видео через браузер — через время получил расшифровку.

## Быстрый старт

```bash
# 1. Собрать и запустить
docker compose up -d

# 2. Открыть браузер
xdg-open http://localhost:8080

# 3. Перетащить видео → получить .txt
```

## Как работает

1. Открываешь http://localhost:8080
2. Кидаешь видео (или аудио) в окно браузера
3. Файл загружается на сервер и становится в очередь
4. На странице виден прогресс: проценты, количество распознанных фрагментов
5. Можно открыть предпросмотр частичной расшифровки
6. Когда готово — кнопка «Скачать текст (.txt)»
7. Файл автоматически удаляется из `input/` после обработки
8. Все готовые расшифровки доступны в разделе «Готовые файлы» — даже после перезагрузки страницы

## Поддерживаемые форматы

**Видео:** MP4, MKV, MOV, AVI, WebM, M4V, TS, FLV, WMV, 3GP, MPG, MPEG  
**Аудио:** WAV, MP3, M4A, OGG, FLAC, AAC, WMA, Opus, AIFF  
**Максимальный размер файла:** 10 ГБ

## Параметры

Переменные окружения (можно указать в `.env` или `docker compose up -e`):

| Переменная | По умолч. | Описание |
|-----------|-----------|----------|
| `WHISPER_MODEL` | `base` | Размер модели: tiny / base / small / medium / large-v3 |
| `WHISPER_LANG` | `ru` | Язык распознавания |
| `WHISPER_DEVICE` | `cpu` | cpu / cuda |
| `WHISPER_COMPUTE` | `int8` | int8 (cpu) / float16 (cuda) |
| `MAX_CONCURRENT` | `1` | Сколько файлов обрабатывать одновременно |

## Модели

| Модель | RAM | 8 ч видео | Качество |
|--------|-----|-----------|----------|
| `tiny` | ~0.3 ГБ | ~2-3 ч | ⭐ |
| `base` | ~0.5 ГБ | ~4-6 ч | ⭐⭐ |
| `small` | ~1 ГБ | ~6-10 ч | ⭐⭐⭐ |
| `medium` | ~1.5 ГБ | ~12-20 ч | ⭐⭐⭐⭐ |
| `large-v3` | ~3 ГБ | ~25-35 ч | ⭐⭐⭐⭐⭐ |

Модель скачивается один раз при первом запуске и кэшируется в volume `whisper-models`.

## Очередь

Обрабатывается один файл за раз. Остальные ждут в очереди. На странице отображается позиция в очереди.

## Прогресс и предпросмотр

Во время обработки на странице отображается:
- **Проценты** — сколько аудио уже распознано
- **Счётчик фрагментов** — сколько段ов расшифровано
- **Предпросмотр** — частичный текст обновляется каждые 5 секунд
- **Последний фрагмент** — показывается в статусе

## Результаты

Для каждого видео создаётся один файл: `output/имя_видео.txt`

Готовые файлы не удаляются — их можно скачать в любой момент через интерфейс (раздел «Готовые файлы»).

## Перенос на другой сервер

```bash
# Сохранить образ
docker save video-transcriber | gzip > video-transcriber.tar.gz

# На сервере
docker load < video-transcriber.tar.gz
docker compose up -d
```

Сервер с NVIDIA GPU:

```bash
WHISPER_DEVICE=cuda WHISPER_COMPUTE=float16 docker compose up -d
```
